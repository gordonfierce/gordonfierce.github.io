Dvorak Drawbacks

Mon 06 April 2015

I started learning Dvorak in 2012, and had made the transition from Qwerty to Dvorak by 2013. However, my QWERTY skills soon atrophied. It might be handy to be able to type in a popular-but-suboptimal layout, so I thought I'd re-learn QWERTY the same way that I learned Dvorak, with TypeRacer.

The interesting question, though, is how quickly I can re-learn it compared with the (incomplete) data I have from when I learned Dvorak.
